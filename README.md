# naoTimes Web Panel
A web panel for naoTimes Showtimes Module

## Development

This project use [pnpm](https://pnpm.js.org/)

```bash
pnpm install
pnpm run dev
```

You can use `openssl rand` to generate a new TOKEN_SECRET<br>
```bash
openssl rand -hex 32
```
